RtlGenRandom import library

This is an import library for RtlGenRandom from advapi32.dll and instructions to reproduce it.

Downloads

These import libraries were made with LLVM 18.1.6 for Windows x64.

Arch Download
x86 rtlgenrandom-x86.lib
x64 rtlgenrandom-x64.lib
arm64 rtlgenrandom-arm64.lib

How to reproduce

Download LLVM for Windows.

Create a text file with these contents and save it as rtlgenrandom.def:

LIBRARY   ADVAPI32
EXPORTS
   SystemFunction036

Run these commands:

llvm-dlltool -d rtlgenrandom.def -l rtlgenrandom-x86.lib -m i386
llvm-dlltool -d rtlgenrandom.def -l rtlgenrandom-x64.lib -m i386:x86-64
llvm-dlltool -d rtlgenrandom.def -l rtlgenrandom-arm64.lib -m arm64
